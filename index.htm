CSS3 transitions

An element change change from one state to another.

.square { border-radius: 0; background-color: rgba(119, 119, 170, 1); transition: all .5s linear; } .square:hover { border-radius: 50%; background-color: rgba(255, 19, 170, 1); }

CSS3 keyframes

An element can change along a timeline.

.keyframes .square { border-radius: 0; animation: roundoff 3s infinite linear; } @keyframes roundoff { 0%, 100% { border-radius: 0; } 50% { border-radius: 50%; transform: translateX(200px); } }

This is a simple demo to show how to set the transform origin of some curved, rotating text.

The transform-origin of the curved text is set halfway across on the Y axis and 240% down on the X (roughly the center of the circle.)

The next step is to make this scalable. It's probably easier to absolutely nest the text box in the circle, so calculating the center point just takes some radius math.

I've used an arbitary asset for this, but would used a more precise path and image width if it needed to be perfect.
